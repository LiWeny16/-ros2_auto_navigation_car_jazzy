## 📊 系统概览

### 模块关系图
```
┌─────────────────┐    ┌─────────────────┐    ┌─────────────────┐
│   仿真模块      │───▶│   感知模块      │───▶│   规划模块      │
│ auto_simulation │    │ auto_perception │    │ auto_planning   │
└─────────────────┘    └─────────────────┘    └─────────────────┘
         │                       │                       │
         ▼                       ▼                       ▼
    地图数据                 障碍物信息               规划路径
         │                       │                       │
         └───────────────────────┼───────────────────────┘
                                 ▼
                    ┌─────────────────┐
                    │   控制模块      │
                    │  auto_control   │
                    └─────────────────┘
                             │
                             ▼
                        控制指令
```

### 技术栈
- **操作系统**: Ubuntu 22.04
- **中间件**: ROS2 Humble
- **通信**: FastDDS + MQTT
- **可视化**: RViz2
- **编程语言**: C++17
- **构建系统**: CMake + Colcon

### 性能指标
- **规划时间**: <1s (A*), <3s (Hybrid A*)
- **控制精度**: 横向误差 <0.1m
- **系统延迟**: <100ms
- **内存使用**: <500MB
- **CPU使用**: <30%
